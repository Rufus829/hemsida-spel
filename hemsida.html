📋 Guide: Så här lägger du till nya spel

För att lägga till ett nytt spel, kopiera koden nedan och lägg till i games-arrayen i JavaScript-sektionen:

{
    id: "unikt-id",
    title: "Speltitel",
    category: "action", // eller puzzle, arcade, strategy, etc.
    description: "Kort beskrivning av spelet",
    thumbnail: "🎮", // Emoji eller bild-URL
    url: "spel-mapp/index.html", // Sökväg till spelet
    date: "2024-01-15"
}

Tips: Om spelet ligger i en undermapp (t.ex. /games/mitt-spel/), ange sökvägen som games/mitt-spel/index.html
